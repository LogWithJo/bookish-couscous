# Calculate Restaurant - Restaurant Order System

A modern, responsive restaurant ordering and calculation system that allows customers to browse a menu, add items to their cart, and calculate totals with optional tax.

## Features

### Menu Display
- Interactive menu grid showing all available food items
- Each item displays: name, price, and image
- Visual hover effects on menu items
- Favorite/heart button on each item

### Shopping Cart
- Add items to cart with a single click
- Remove items from cart
- Real-time cart item count in the header
- View all selected items in the cart section
- Individual item quantities displayed

### Price Calculation
- Automatic subtotal calculation
- Optional tax calculation (toggleable)
- Configurable tax rate (default: 12%)
- Real-time total updates

### Additional Features
- Process Order functionality
- Clear Cart functionality
- Responsive design (works on mobile and desktop)
- Sticky cart section for easy access

## How to Use

1. Open `index.html` in any modern web browser
2. Browse the menu items
3. Click the "+" button or "Add to Cart" to add items
4. Click the "-" button to remove items
5. Toggle "Add Tax" to enable tax calculation
6. Adjust tax rate if needed (default 12%)
7. View the total in the cart section
8. Click "Process Order" to complete the order
9. Click "Clear Cart" to reset the order
